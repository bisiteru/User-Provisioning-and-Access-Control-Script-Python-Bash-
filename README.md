# User-Provisioning-and-Access-Control-Script-Python-Bash-
<h2>Description</h2>
Project consists of a simple PowerShell and Python script that walks the user through how to Develop a script (Python or Bash) that automates user provisioning across different systems, Integrate with existing user directories (e.g., Active Directory, LDAP) to retrieve user information, and Implement logic for creating user accounts, assigning appropriate permissions, and disabling inactive accounts.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Python</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
